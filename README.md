# 🎉 Eventify - Event Management System (ASP.NET MVC)

## 📌 Problem Statement  
Build a system to manage event planning, booking, and participation from creation to completion.

---

## 🎯 Objective  
To develop a centralized platform that simplifies event organization, improves booking management, and enhances user engagement.

---

## ✅ Expected Outcome  
A complete event lifecycle system with:
- Event creation & management  
- Booking & tracking system  
- Role-based dashboards  
- Analytics & reporting  

---

## ⚠️ Constraints & Considerations  

- Must support multiple events at the same time  
- Role-based access control (Admin / Organizer / User)  
- Real-time booking updates  
- Accurate and consistent event history  

---

## 📅 Progressive Milestones  

- **T+4:** Event creation + event listing UI  
- **T+8:** Booking system + organizer dashboard  
- **T+16:** Search/filter + analytics + reports  
- **T+24:** Payment integration + notifications + UI polish  

---

## 📌 Project Overview  

**Eventify** is a web-based Event Management System built using **ASP.NET MVC**.  
It enables users to create, manage, and book events with a smooth and interactive interface.

---

## 👥 User Roles  

### 👑 Admin  
- Manage users  
- Monitor all events  
- View analytics & reports  
- Control event listings  

---

### 🎤 Organizer  
- Create & manage events  
- Track bookings  
- View dashboard (sales & visitors)  
- Update event details  

---

### 👤 User  
- Browse events  
- Book tickets  
- View booking history  
- Manage profile  

---

## ⚙️ Features  

- 🎟️ Event Creation & Management

- 🔍 Search & Filter Events
